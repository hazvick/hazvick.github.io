Music Player created by Hazvick

To be able to run the software, you need to install the following.
Prerequisites: - Java Runtime Environment

How to use this software:
1 - Open folder "dist"
2 - Launch MusicPlayerApp.jar
* wait for application to load *
3 - Click button "Random Song" to play a random song
(Click the button again to play a new random song)
4 - Click button "Stop" to stop the music playing
